Python

Python ist eine Programmiersprache die häufig für die Entwicklung von Webanwendungen, sowie Softwareentwicklung,
aber auch für Datenwissenschaft und Oberflächenentwicklung genutzt wird.
Python ist eine der einfacheren Programmiersprachen und ziemlich effizient und leicht zu lernen.
Der Anspruch von Python ist es übersichtliche und gut lesbare Programme zu entwickeln.
Die Programmiersprache Python wurde Anfang der 1990er Jahre von Guido van Rossum entwickelt.
Aktuell gibt es 29 Versionen, wovon 24 nicht mehr unterstützt werden.
